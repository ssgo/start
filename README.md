# Smart Service Go

a start tools for smart service solution

``` shell
curl http://ssgo.cc | sh
```

will install a shell script to /usr/local/bin/ssgo for start use smart service go

``` shell
Usage:
    ssgo
    ssgo i sdk
    ssgo install sdk
    ssgo i net
    ssgo i text
```

windows user please install Cygwin or MinGW first.
